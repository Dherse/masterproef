# Programmable layers for photonic FPGAs

This master thesis focused on finding ways of expressing photonic circuit using code and using it to configure and use photonic processors.

## Building

To build this report, you will need the TeX engine [tectonic](https://tectonic-typesetting.github.io/en-US/). And simply run the following command:

```bash
tectonic -X build
```

Then, the document will be located in `build/master_thesis/master_thesis.pdf`.
